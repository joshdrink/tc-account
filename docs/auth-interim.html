2-factor Authentication

2-factor authentication helps protect your account by adding a second layer of security beyond your username and password. Talent Cloud implements this second layer using a standard that asks for a one-time password when you log into the site. These one-time passwords are set up through apps such as Google Authenticator or Authy, and provide you with the ability to generate the one-time password on demand.
